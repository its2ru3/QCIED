# Fig. 12: Filter Window and Masks of Edge Detection
**Context:** Spatial definition of the improved Sobel operator masks used for calculating gradients.
**Description:** - Unlike quantum circuits, this shows classical matrix representations that the quantum algorithm simulates.
- (a) A 3x3 pixel filter window centered at $P(y,x)$.
- (b-e) Four 3x3 convolution masks corresponding to four directions: 0°, 45°, 90°, and 135°. The matrices contain weights (e.g., -2, -1, 0, 1, 2) which are applied to the local neighborhood to calculate directional gradients.
**Gate Sequence:** N/A (Classical matrix definitions).
