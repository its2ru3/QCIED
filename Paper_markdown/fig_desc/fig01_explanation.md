# Fig. 1: 4x4 Color Image and OCQR Model
**Context:** Representation of a 4x4 color image using the OCQR (Optimized Quantum Representation of Color Images) model.
**Description:** - The image shows a 4x4 grid of pixels with coordinates (Y,X) from 00 to 11. Each pixel holds R, G, B color values (ranging from 0-7).
- Beside the grid is the mathematical quantum state expression $|I\rangle$. 
- The expression demonstrates how intensity, color channel, and spatial location information are entangled in superposition to represent the entire image in a single quantum state.
