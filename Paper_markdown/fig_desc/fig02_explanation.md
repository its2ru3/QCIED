# Fig. 2: 4x4 Color Image and Eight Neighborhood Images
**Context:** Spatial shifting for neighborhood operations (like convolution or edge detection).
**Description:** - Displays a central 4x4 color image (labeled Y, X).
- Surrounding it are eight shifted versions of the same image representing the spatial neighborhood (from top-left $Y+1, X-1$ to bottom-right $Y-1, X+1$).
- This illustrates the required data state to perform local quantum image processing operations across a 3x3 pixel window.
